1. Introduction

Empire Backgammon est développé par [TON NOM / TA SOCIÉTÉ]. Cette politique décrit comment nous traitons vos données lorsque vous utilisez notre application.

En résumé : Empire Backgammon est un jeu hors ligne. Nous ne collectons aucune donnée de jeu. Seul Google AdMob collecte des identifiants publicitaires, que vous pouvez refuser.

2. Données collectées

Par nous directement : aucune. Pas de compte, pas de serveur, pas de données transmises.

Vos statistiques et préférences sont stockées uniquement sur votre appareil.

Par nos partenaires tiers :

  • Google AdMob — identifiant publicitaire (GAID Android / IDFA iOS), interactions publicitaires, IP approximative. Politique Google →
  • Google Play Billing — traitement du paiement pour l'achat "Supprimer les publicités". Nous ne recevons pas vos données bancaires.

3. Utilisation des données

Les données AdMob servent uniquement à :

  • Afficher des publicités pertinentes (version gratuite)
  • Mesurer la performance publicitaire

Si vous achetez la suppression des publicités (0,99€), aucune donnée publicitaire n'est plus collectée.

4. Vos droits (RGPD)

  • Accès — accéder à vos données
  • Rectification — corriger des données inexactes
  • Effacement — demander la suppression
  • Opposition — refuser le traitement publicitaire

5. Contrôle des publicités

Android : Paramètres → Google → Annonces → Désactiver la personnalisation

iOS : Paramètres → Confidentialité → Publicités Apple

Dans l'app : Paramètres → Paramètres Publicitaires

6. Enfants

L'app est classée PEGI 3. Nous ne collectons sciemment aucune donnée d'enfants de moins de 13 ans.

7. Sécurité

Toutes les communications réseau utilisent HTTPS/TLS. Les données locales sont dans le stockage sécurisé Android (AsyncStorage).
